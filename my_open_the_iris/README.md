# Welcome to My Open The Iris
***

## Task
The goal of this project is to analyze the Iris dataset and provide a clear summary of its structure and content.
The main challenge is to format the output exactly as expected, including dataset dimensions, sample rows, statistical summaries, and class distribution.

## Description

The problem is solved by loading the dataset and using data analysis techniques to extract meaningful information.
The project prints the dataset shape, displays the first rows, generates descriptive statistics, and shows the distribution of classes using efficient and readable methods.

## Installation

Clone the repository and install the required dependencies before running the project.
Make sure Python and the necessary libraries are installed.

Example:

pip install -r requirements.txt

## Usage

Run the program to display a full summary of the dataset in the terminal.

Example:

./my_project argument1 argument2
